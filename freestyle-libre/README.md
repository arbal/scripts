# Usage Guide

#### Options

``` 
./libre_app.pl --help
Usage: libre_app.pl [options]
  where options are: 
   --import <filename> --type <liapp|libre>
   --export <filename>
   --db <dbname> 
   --months <year> 
   --weeks <numberofweeks> 
   --days <numberofdays> 
   --help usage
```
